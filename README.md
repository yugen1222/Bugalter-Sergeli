# Материальный бухгалтер v4 — бот и сайт в одном сервисе

## Исправления

- первая, вторая и третья смены одного сотрудника за один день сохраняются отдельно;
- повторное открытие той же смены больше не удаляет прежние позиции;
- старая база автоматически мигрирует на новую схему без удаления данных;
- дневной отчёт показывает все смены;
- авторизация сайта работает через красивую форму входа внутри Telegram;
- интерфейс сайта обновлён для телефона и компьютера;
- в боте добавлен пошаговый индикатор заполнения.

## Render

Используется один Web Service.

Build Command:

```bash
pip install -r requirements.txt
```

Start Command:

```bash
python combined_app.py
```

Переменные:

```env
BOT_TOKEN=...
TIMEZONE=Asia/Tashkent
ALLOWED_USER_IDS=
ADMIN_USER_IDS=
REMINDER_TIME=20:00
WEB_USERNAME=admin
WEB_PASSWORD=ваш_пароль
SECRET_KEY=длинная_случайная_строка
PYTHON_VERSION=3.13.5
```

После загрузки файлов:

1. Render → Manual Deploy.
2. Clear build cache & deploy.
3. Откройте сайт.
4. Появится обычная форма входа вместо надписи «Требуется авторизация».
