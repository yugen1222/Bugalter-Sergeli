# Материальный бухгалтер v5

## Новая логика расследования

Каждая минусовая позиция — это один случай ошибки, независимо от количества товара.
Например, минус 6 булочек с причиной «Неправильная приёмка» считается как:

- 1 случай ошибки;
- 6 единиц товара в минусе.

После количества бухгалтер выбирает несколько выполненных контрмер:

1. Проверка выравнивания
2. Проверка движения товара
3. Сверка движения
4. Проверка закрытости накладных
5. Сверка движения с накладной
6. Проверка приёмки товара по видео
7. Проверка склада остатков

Затем выбирается одна итоговая причина.
