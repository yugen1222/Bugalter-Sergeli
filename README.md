# Материальный бухгалтер v6

## Главное изменение

Каждая минусовая позиция имеет отдельный статус:

- 🔴 Открытая позиция — проблема ещё не устранена и продолжает считаться текущим минусом.
- 🟢 Закрытая позиция — проблема устранена, но остаётся в истории и аналитике.

## Как работает бот

После товара, количества, контрмеры, причины, ответственного и комментария бот спрашивает:

- оставить позицию открытой;
- закрыть позицию сразу.

Открытую позицию позже можно закрыть:

- в Telegram через кнопку «🔴 Открытые позиции»;
- на сайте через кнопку «Закрыть».

При закрытии можно указать, как проблема была устранена.

## Сайт

На сайте есть два окна:

1. Открытые позиции.
2. Закрытые позиции.

Раздел «Товары с минусами» показывает только открытые позиции.

Также добавлена динамика по дням:

- сколько новых минусов зарегистрировано;
- сколько позиций закрыто.

Это позволяет наблюдать, уменьшается ли количество активных минусов и насколько эффективно бухгалтер закрывает найденные случаи.
